|===
^|*Recommendation {counter:rec-id}* |*/rec/message-encoding/node*
^|A |A Node message ``name`` SHOULD contain a human-readable identification of the device, such as its model name or type designation.
^|B |A Node message SHOULD define a ``location`` element to describe the geographic location of the Node device.
