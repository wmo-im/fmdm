[[req_data-model-standard_name_binding_hierarchy]]
[width="90%",cols="2,6a"]
|===
^|*Requirement {counter:req-id}* |*/req/data-model/standard_name_binding_hierarchy*
^|A |Standard names SHALL be drawn in the following order, in the following order of precedence:

* Level 1: CF Metadata Conventions
* Level 2: WMO First Mile parameter registry
* Level 3: Locally managed vocabularies

^|B |The CF namespace prefix SHALL be declared using the value ``cf``.
^|C |The CF namespace URI value SHALL be a URI that resolves to a machine-readable vocabulary server endpoint publishing the CF standard name table.
^|D |The WMO First Mile registry namespace prefix SHALL be declared using the value ``wmo``.
^|E |Locally managed vocabularies SHALL conform to the FAIR Guiding Principles for scientific data management (Findable, Accessible, Interoperable, Reusable). In particular:
^|F |The vocabulary SHALL be findable via a persistent, unique identifier (the namespace URI).
^|G |The vocabulary SHALL be accessible at that URI to all parties participating in the data exchange.
^|H |The vocabulary SHALL use an open, standardised representation (e.g., SKOS/RDF).
|===
