[[req_message-encoding-observation]]
[width="90%",cols="2,6a"]
|===
^|*Requirement {counter:req-id}* |*/req/message-encoding/observation*
^|A | An Observation message SHALL define a ``parameterDefinitionId`` element using the id of the ParameterDefinition message.
^|B | An Observation message SHALL define a ``time`` element using ``google.protobuf.Timestamp``, in UTC.
^|C | An Observation message SHALL define one or more ``values`` elements using the Value message.
|===
